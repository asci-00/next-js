## BACK-END

> node-js 와 express를 활용한 api server 구현 

## `sequelize`

- sequelize를 통해 데이터베이스를 직접 서버에 CREATE 할 수 있음
- models에 CREATE할 TABLE를 설계해놓고 해당 model을 db에 적용

### `nodemon`

- node app.js 명령은 핫 리로딩 기능을 제공하지 않기 때문에, nodemon을 통해 핫 리로딩 기능 적용

```bash
nodemon app.js
```
